# APIs & External Services - Intuitives

## Browser APIs
- **Web Audio API:** The core sound engine.
- **Web MIDI API:** For connecting physical keyboards/controllers.
- **Canvas / WebGL:** For the visual interface.

## AI Libraries
- **Magenta.js:** Google's open-source library for AI music generation in the browser.
- **MediaPipe:** For robust, low-latency hand tracking via webcam.
